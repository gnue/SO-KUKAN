USBeep

On 2011年1月5日, in , by gnue

Download

USBデバイスを挿したときおよび抜いたときに音声で知らせてくれるジョークソフト。

Download

The voice tells us when USB device inserted and removed.
This is joke software.

インストール

  • USBeep のファイルをダブルクリックするとシステム環境設定にインストールされます

設定

  • システム環境設定で「使用する」をチェック
  • 「読み上げる声」を選択
  • 挿入・削除時の台詞（英語のみ）を入力
  • ※ Mac OS X 10.7 Lion では「Kyoko」を選択すると日本語の台詞も可


Install

  • USBeep file is double-clicked, it is installed in the System Preferences

Preferences

  • In System Preferences “Enable” check
  • Select Voice
  • Enter the words (English only)
